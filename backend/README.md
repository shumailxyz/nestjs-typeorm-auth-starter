# nestjs-typeorm-auth-starter

## Description

Simple starter kit for Nest.js that includes typeorm & auth setup. Once the app is running, you can navigate to `localhost:3000/api-docs` to see available API routes.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
